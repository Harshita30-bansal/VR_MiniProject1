# VR_MiniProject1: Multi-Object Apparel Detection and Instance Segmentation

### Objective:
Detect, classify, and segment multiple clothing items in a single image using deep learning, based on visual attributes such as garment type, shape, texture, and spatial location within the image.

### Overview:
This is a multi-task visual recognition problem built on the DeepFashion2 dataset. The project performs the following two tasks across the top five most frequently occuring clothing categories: 
- Multi-Label Classification -- Models used : _ResNet-50, EfficientNet, MobileNetV3_
- Bounding Box Detection and Pixel-Level Instance Segmentation -- Models used : _YOLO, Mask R-CNN, U-Net_
